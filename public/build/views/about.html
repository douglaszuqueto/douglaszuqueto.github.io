Sobre o Blog

Esse blog foi criado com o objetivo de compartilhar um pouco do meu conhecimento e experiencia para a comunidade através de projetos que venho desenvolvendo. Terá como foco inicial o tema Internet das Coisas", onde pretendo criar desde simples aplicações , até projetos mais complexos usando a Plataforma Arduino e ESP8266.

Segue abaixo alguns dos temas que pretendo abordar aqui no Blog:

  • Ambiente de Desenvolvimento para Internet das Coisas
  • Bibliotecas Utilizadas
  • Protocolos
  • Hardware Utilizado
  • Desenvolvendo a primeira aplicação para IoT
  • As camadas que compoem uma aplicação IoT
  • Projetos mais completos(Hardware, Software, Aplicação e Banco de Dados)
